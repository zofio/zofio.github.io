本地部署nexus3 搭建maven私服

2022-03-04

由于JFREG的一系列惊人操作，GITEE关闭maven功能，不得已，本地搭建以下maven服务。

准备工作：JDK （如果你的设备存在JDK环境，请忽略这部分。）

安装JDK8

1.下载

下载地址：https://www.sonatype.com/thanks/repo-oss 选择你的系统版本进行下载。这里演示的是linux版本

2.配置

nexus默认占用端口为8081，如果端口已经被使用，需要在nexus-3.xxxx/etc/nexus-default.properties 文件中进行修改

3.运行/安装

下载好之后解压，得到nexus-3.xxxx，进入之后有两个文件夹，启动终端，进入nexus-3.xx，进入bin，然后运行

windows需要等待如之输出提示成功，linux在提示成功后等待1-2分钟，打开你的本地站点即可。

4.登陆

第一次登陆，会提示你密码保存在nexus-3.32.0-03-unix/sonatype-work/nexus3/admin.password 文件中，自己打开复制密码即可。账号是默认的admin。登陆之后会提示你修改密码和是否打开匿名访问，建议打开。

5.配置和上传

进入之后点击左侧的upload，然后点击右侧的maven-release，然后按照下面的信息填写就可以了。
